# 装机必备

## 软件安装

- Chrome

- 迅雷，用来下载其他下载比较慢的应用

- nvm

- 通过 nvm 安装 node 和 npm `nvm install 16.14.0`，mac 用 n

- 通过 npm 安装 nrm `npm install nrm`，管理 npm 源

- git

- VSCode

- WPS、Typora、Postman

## Chrome 插件

- google 上网助手，免费梯子，通过安装包加载，用来下载其他插件

- FeHelper，很好用的前端助手

- 沙拉查词，划词翻译和页面翻译

- Mokku，mock 数据插件

- React、Vue 调试工具
